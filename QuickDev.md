## Pricing

All packages include a dedicated point of contact, clear milestone-based delivery, and full source code ownership on handoff. No recurring agency fees unless you opt into a retainer.

---

### Starter — $5,000

**Best for:** Landing pages, business websites, and simple web apps with limited scope.

**Includes:**
- Up to 5 pages / core screens
- Frontend design + development (React or plain HTML/CSS)
- Basic contact or lead capture form
- Mobile-responsive layout
- Deployment setup (Vercel, Netlify, or similar)
- 1 round of revisions post-delivery
- 2 weeks delivery target

**Not included:** Backend/database, auth systems, third-party API integrations

---

### Growth — $6,999

**Best for:** Full-stack web apps with user auth, a database, and a small set of core features.

**Includes:**
- Everything in Starter
- Backend API development (Node.js / Next.js API routes)
- Database design and setup (PostgreSQL / Supabase)
- User authentication (email/password, OAuth)
- Up to 3 third-party integrations (e.g. Stripe, email, maps)
- Admin panel or basic dashboard
- 2 rounds of revisions post-delivery
- 3–4 weeks delivery target

**Not included:** Multi-tenant architecture, complex role systems, mobile apps

---

### Pro — $7,500

**Best for:** SaaS platforms, client portals, and multi-role applications that need production-level architecture.

**Includes:**
- Everything in Growth
- Multi-role access control (admin, staff, client, etc.)
- Multi-tenant or multi-location data architecture
- Stripe subscription or one-time payment integration
- Notification system (email or in-app)
- CI/CD pipeline setup
- Basic analytics or reporting dashboard
- 3 rounds of revisions post-delivery
- 4–5 weeks delivery target

---

### Enterprise — $10,000

**Best for:** Complex, high-traffic, or mission-critical platforms requiring custom architecture, deep integrations, or long-term scalability planning.

**Includes:**
- Everything in Pro
- Custom system architecture and technical planning session
- Advanced third-party integrations (ERP, logistics APIs, CRMs, etc.)
- Performance optimization and load testing
- Full documentation (API docs, deployment runbook, codebase overview)
- Security review (auth hardening, input validation, OWASP checklist)
- 30 days of post-launch support
- Flexible timeline scoped to project complexity

**Also available as a retainer** for ongoing development, feature additions, and maintenance.
